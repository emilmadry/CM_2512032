
# Playwright

This is a basic playwright test suite


## Requirements

Node.js, npm

## Installation

Install dependencies

```bash
  npm install
```
    
## Running Tests

To run tests, run the following command

```bash
  npx playwright test
```
To run them in UI mode use
```bash
  npx playwright test --ui
```
To run test group, e.g. smoke, run
```bash
  npx playwright test --grep @smoke
```
